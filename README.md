# Tic-Tac-Toe
Tic Tac Toe game in cpp


--------------------------------------------------------------------
Compiling in linux

Requirement:g++ complier that support c++14 and cmake

    $sudo apt install g++
    $sudo apt install cmake

$cd TIC-TAC-TOE
$make
-----------------------------------------------------------------------

------------------------------------------------------------------------

Compiling in windows with MinGW

$cd TIC-TAC-TOE
$g++ main.cpp ui.cpp tictactoe.cpp board.cpp player.cpp -o TicTactoe -std=c++14
